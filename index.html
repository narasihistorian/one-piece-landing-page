ABOUT THIS PROJECT

Hey Gretting, let me introduce you into my project. Last weeks I found in the instagram @dailywebdesign about competition to build web Design with anime theme. So based on the anime genre, I decide to choose my favorite anime, One Piece as a theme which is I used into my Project. The storytellling about my project is simple, I just tell main character about Straw Hat crew which is main Character in the One piece World.

My main tools for this project, I've using Figma to design wireframe and the structure my project, then I'm convert my design prototype into real world project using pure HTML, CSS, and javascript, GSAP and BarbaJS library to create interactive animation on my project.

My style in this project I'm using Grid box mixed page transition for better user experience and put interactive, simple and visual pleasing assets which already responsive on fullHD (1920px), desktop (1280px), Tab (1024px) and Mobile (720px). Less perfect actually, stil have minor bug but Hopely you enjoy with my project.
